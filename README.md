# 系统介绍

本系统为股票交易预测系统，总共有三个模块组成：

1.股票数据管理模块

本模块实现了可视化表格页面，可视化K线图以及股票的下载保存。

2.用户管理模块

可以实现用户的登录。

3.预测模块（核心功能）

可以通过输入股票代码自动爬取股票近期的多维数据，并通过用户的选择对数据的处理进行ARIMA预测和LSTM预测。生成明日该股票的价格数值。
